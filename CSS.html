CSS

종속형 시트 또는 캐스케이딩 스타일 시트(Cascading Style Sheets, CSS)는 마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어(style sheet language)로, HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다. W3C의 표준이며, 레이아웃과 스타일을 정의할 때의 자유도가 높다. 기본 파일명은 style.css이다.
마크업 언어(e.g. HTML)가 웹사이트의 몸체를 담당한다면 CSS는 옷과 액세서리처럼 꾸미는 역할을 담당한다고 할 수 있다. 즉, HTML 구조는 그대로 두고 CSS 파일만 변경해도 전혀 다른 웹사이트처럼 꾸밀 수 있다.
현재 개발 중인 CSS3의 경우 그림자 효과, 그라데이션, 변형 등 그래픽 편집 프로그램으로 제작한 이미지를 대체할 수 있는 기능이 추가되었다. 또한 다양한 애니메이션 기능이 추가되어 어도비 플래시를 어느 정도 대체하고 있다.

기초 사용법

  • 선택자(Selector)를 이용해 특정 태그에만 지정된 효과(Declaration, 선언)를 적용 가능.
    • 선언 내부는 속성(Property)와 값(Value)으로 형성됨.
  • style 태그를 head 태그 안쪽에 적고 중괄호를 이용해 디자인을 결정함.
  • 콜론(:)을 특정 선택자 뒤에 붙여서 가상 선택자(:) 및 가상 요소(::)를 지정할 수 있다.

박스 모델

  • 태그의 모양은 block model element와 inline element라는 박스 모델로 형성되어 있다. display 속성을 통해 박스 모델의 값을 변경할 수 있다.

그리드

  • 'display:grid;'를 통해 선언하고, grid-template-column 속성을 통해 자식 태그들을 같은 열에 배치할 수 있다. 그 후, 각 자식 태그의 길이를 순서대로 지정할 수 있으며, px or % 값으로 고정된 길이로, fr 값을 통해 특정 비율로 설정할 수 있다.

선택자의 종류

  • a
  • .(Full stop): 특정 class의 태그를 선택. 일반 선택자보다 우선순위가 높고 뒤에 적을수록 우선순위가 높다.
  • #: 특정 id의 태그를 선택. '.(class 선택자)'보다 우선순위가 높다.
  • ':hover': 해당 요소에 커서를 가져다 댄 경우를 선택.
  • ':active': 해당 요소를 클릭한 경우를 선택.

속성과 값의 종류

  • display: 해당 박스 모델의 표시 방식을 설정.
    • block: 해당 태그의 박스 모델을 블록으로 설정.
    • inline: 해당 태그의 박스 모델을 인라인으로 설정.
    • none: 해당 태그를 안 보이게 설정.
    • grid: 해당 태그를 그리드화.
  • padding: border와 content 사이의 간격을 설정. border 바깥의 간격을 설정하는 margin과는 다르다.
  • width: (주로 박스 모델의) 너비를 설정.
  • grid-template-column: 자식 태그를 같은 열에 배치.
    • fr: fraction. 분수. 고정값으로 선택한 뒤 남은 너비의 비율을 설정함.
      • e.g. 전체 너비가 A라고 했을 때, '100px 1fr 2fr'로 설정하면, 100px, (A-100px)*1/3, (A-100px)*2/3가 된다.

노트

  • block level element와 inline element 차이를 이해하고 적절히 사용하기.
  • <!---->(HTLM), /**/(CSS)를 이용하여 주석 달기.
  • 동일 속성 및 값을 공유하는 선택자는 ',(콤마)'를 이용해서 묶을 수 있음.
  • 속성에서 -(하이픈)으로 구분짓는 세부 속성은, CSS에서 띄어쓰기로 구분하여 연속으로 설정할 수 있다.
