HTML

HTML(Hypertext Markup Language)은 웹 페이지를 만드는 데에 쓰이는 표준 마크업 언어이다.

목차

  1. 태그의 종류
  2. 태그의 속성
  3. 노트

HTML에서의 태그 사용법.
<,>사이에 태그를 입력. 닫을 때는 태그 앞에 슬래시(/) 추가

태그의 종류


strong: 강조.
u: Underline. 밑줄.
h1 ~ h6: Heading. 헤딩(제목). 굵게 표시되며, 숫자가 높아질 수록 글자가 작아짐. 자동으로 줄바꿈이 일어남. 6까지만 가능.
br: Line Breaker. 줄바꿈. 닫을 필요가 없으며 중복 사용을 통해 여백의 길이 조정 가능.
p: Paragraph. 문단. br과 마찬가지로 줄바꿈이 일어나며 슬래시로 닫을 수 있음. 문단이 어디서부터 어디까지인지를 나타내줄 수 있다는 게 특징.
img: Image. 이미지 삽입. 한 칸 뒤에 속성(Attribute)을 삽입. 주소는 src(source)를 사용한다. width를 통해 사이즈를 조절할 수 있으며, %로도 조절이 가능하다.
li: List item. 목록. ul 태그의 자식(child) 태그.
ul: Unorderd list. li 태그의 부모(parent) 태그.
ol: Orderd list. li 태그의 부모(parent) 태그. 자식 태그에 숫자를 차례대로 부여.
title: 웹 페이지의 제목을 지정.
meta: 메타데이터. charset 속성을 이용해 utf-8을 입력하면 한글이 깨지지 않고 정상적으로 출력됨.
head, body: 본문(body)과 본문을 설명하는 부분(head)을 구별하는 태그.
html 헤드와 바디를 감싸는 태그.
!DOCTYPE html 문서 최상단에 기재하여 HTML 문서임을 선언.
a: Anchor. 링크 태그. 문서 내부 혹은, 외부로 이동. href 속성을 통해 원하는 부분으로 이동. 속성 target="_blank"를 이용해 새창으로 이동이 가능. 속성 title=""을 이용해 툴팁을 표시. (e.g. 생활코딩)
i: Italic. 기울임꼴.
del: delete. 삭제된 텍스트. 취소선을 표시.
sup, sub: Superscript, Subscript. 위첨자, 아래첨자.
small: 텍스트를 작게 표시.
abbr: Abbreviation. 약어를 표시. 속성 title을 이용한다. (e.g. HTML)
charset: Character sets.
span: 특정 구역에 스타일을 지정할 때 사용. inline element.
div: Divison. span 태그와 비슷하나 줄넘김이 있고 사각형 박스 모양으로 구성. block level element.

style: CSS언어로 디자인 함을 나타냄. head 태그 안에 쓰며, 원하는 태그를 지정하여 꾸밀 수 있고, 이때 중괄호('{}')를 사용.


태그의 속성


style: 태그 요소를 꾸며주는 속성.
  • (background-)color
    • : 색상 및 색상 코드를 입력. RGB는 괄호를 뒤에 붙이며 HEX는 앞에 #을 붙인다.
  • margin
    • : 여백 입력. 위치(e.g. left)와 정도(px 단위)를 붙인다.
  • font-size
    • : 글자 크기. 정도(px 단위)를 뒤에 붙인다.
  • white-space
    • : 요소의 공백 방식을 정의한다. pre를 붙이면 띄어쓰기를 실제 적용한 만큼 나타난다.
  • text
    • : 텍스트의 요소를 꾸며준다. 뒤에 하이픈(-)을 붙여서 사용하며, align:'방향'(정렬), decoration:none(글자 꾸밈 삭제)등이 있다.

href: Hypertext Reference. Anchor 태그 뒤에 붙어서 링크의 목적지를 설정.
target:
title: 텍스트에 툴팁을 표시.
src: source.
charset:
id: 태그에 고유한 값을 부여.
pre:
class: 태그들을 그룹짓는 속성. 여러 개가 가능하며 띄어쓰기로 구분.

노트

  • 제목을 작성할 때 크기와 굵기를 따로 입력하는 것보다 h를 사용하는 것이 검색 엔진을 이용한 결과 검색에서 더 상위에 놓일 수 있다.
  • 태그 뒤의 속성을 활용하자.
  • Alt(VS Code 기준)을 누른 상태에서 커서를 다른 곳에 찍으면 동시에 입력 가능.
  • 리스트는 부모(ol, ul)와 자식(li) 태그로 구분할 수 있다. 부모 태그는 자식 태그를 들여 쓰게 한다. ul은 동일 기호, ol은 숫자를 차례대로 자식 태그에 부여한다.
  • 웹 페이지 최상단의 제목을 title 태그로 지정할 수 있고, 한글이 깨질 시에는 meta 태그와 charset 속성으로 utf-8을 지정해주면 정상적으로 나온다. 이러한 본문을 설명하는 부분은 head 태그로 묶고, 본문은 body 태그로 묶는다.
  • head 위와 body 아래에는 html 태그로 묶는다.
  • 크롬 기반 브라우저에서 Ctrl+Shift+C를 이용하여, 개발자 도구의 요소 검사를 진행함으로써 웹의 구성을 볼 수 있다.
